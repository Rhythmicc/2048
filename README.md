# 2048 Game

## Build

```shell
gcc -O3 main.c -o cmake-build-debug/2048
./cmake-build-debug/2048 -a # Auto Play
./cmake-build-debug/2048 -a # Play with suggestion
```

## Build With Qpro

```shell
qrun -br -a # build and auto play
qrun -br -s # build and play with suggestion
```
